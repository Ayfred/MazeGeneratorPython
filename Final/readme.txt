The goal of this assignment is to implement the following algorithms:
    - Breadth First Search (Bfs)
    - Depth First Search (Dfs)
    - A* algorithm
    - Value Iteration
    - Policy Iteration

------------------------------------------------------------------------------------------------------------------------
This is how to run the program:
    1. Open the terminal
    2. Go to the directory where the Python file is located
    3. Type `python Main.py` followed by the option number and 'y' if you want to animate the process
    4. Press Enter

------------------------------------------------------------------------------------------------------------------------
Options:
    -1: Bfs
    -2: Dfs
    -3: A*
    -4: Value Iteration
    -5: Policy Iteration
    -6: Comparison between all algorithms (Bfs, Dfs, A*, Value Iteration, and Policy Iteration) :
        choose the three numbers of the size of the maze

------------------------------------------------------------------------------------------------------------------------
Example:
    'python Main.py 3 y'
